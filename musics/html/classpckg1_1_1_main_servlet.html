Main part of the project is here. Whenever a post action is required, program comes to this method. Post actions are after login page, after search button, after show saved entries, after save selected entries. And there is one get action for returning login page. This method handles creating and running html code of the functions that are listed above. And so this method is the main user interface of my project.
